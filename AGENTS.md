- Moda elegante e sofisticada

- Peças exclusivas: vestidos, blusas, acessórios
- Posicionamento: premium acessível

## 🎨 Design System

### Cores
```css
--primary: #1a1a1a      /* Preto elegante */
--secondary: #c9a87c    /* Dourado suave */
--accent: #8b7355       /* Marrom terreo */
--light: #fafafa        /* Off-white */
--gray: #6b6b6b         /* Cinza texto */
--border: #e5e5e5       /* Bordas sutis */
```

### Tipografia
- **Display**: Playfair Display (títulos)
- **Body**: Poppins (texto)

### Elementos UI
- Espaçamentos generosos
- Bordas minimalistas
- Sombras sutis
- Transições suaves

## 📝 Notas para IA

### Como Trabalhar neste Projeto

1. **Respeite o design system** - cores, tipografia e espaçamentos definidos
2. **Mobile-first** - sempre pense responsivo
3. **Acessibilidade** - use HTML semântico, alt texts, contrast
4. **Performance** - minimize assets, otimize imagens
5. **Manutenibilidade** - código limpo, comments onde necessário

### Padrões de Commit
```
feat: nova funcionalidade
fix: correção de bug
style: ajustes visuais
docs: documentação
refactor: refatoração
```
